# Miguel Bonilla — Portfolio

Portfolio personal desarrollado con **Next.js 16**, **Mantine UI** y **Three.js**. Presenta proyectos, habilidades y experiencia profesional como Ingeniero de Sistemas Full Stack.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** Mantine v8, Tailwind CSS v4
- **3D:** Three.js + React Three Fiber
- **Animaciones:** Motion (Framer Motion)
- **Lenguaje:** TypeScript
- **Fuentes:** Syne, JetBrains Mono

## Inicio rápido

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Servidor de desarrollo |
| `npm run build` | Build de producción |
| `npm run start` | Servidor de producción |
| `npm run lint` | Linter ESLint |

## Despliegue

El proyecto está optimizado para desplegarse en [Vercel](https://vercel.com).
